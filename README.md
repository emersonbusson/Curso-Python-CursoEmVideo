## Trata-se de um total de 115 desafios &#x1F40D;
### A maioria dos exercícios vão ter outros **arquivos ".py"** com as seguintes informações:

* Desafio 095, Aula 19,**forma 01**.py
* Desafio 094, Aula 19,**forma 02**.py
